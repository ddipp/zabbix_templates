# Template for Huawei UPS2000H

The template is designed for a stacked UPS.

## Data collection:
- Input voltage, current, and frequency data collection;
- Battery data collection: voltage, charge percentage, and estimated battery runtime;
- Load data collection: voltage, current, active and apparent power;
- Temperature of each unit in the stack;

## Triggers (thresholds are set using macros):
- Overtemperature;
- Undervoltage;
- Input voltage loss;
- High load (%).
